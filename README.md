# InkPad

Bloco de notas com **agenda** e **SQL** — simples, rápido e otimizado para iPhone.

## O que tem

- **Anotações** — texto livre com busca e fixadas
- **Agenda** — calendário mensal + compromissos por dia
- **SQL** — snippets com highlight, copiar e categorias
- **Tipos** — Anotações, Pessoal, Agenda, SQL
- **Offline** — dados no aparelho (IndexedDB)
- **PWA** — adicione à tela de início no iPhone

## Stack

- Next.js (App Router)
- Tailwind CSS
- Dexie (IndexedDB)
- Deploy na Vercel

## Desenvolvimento

```bash
npm install
npm run dev
```

Abra [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Deploy (Vercel)

1. Crie o repositório no GitHub
2. Importe no [vercel.com/new](https://vercel.com/new)
3. Deploy — framework Next.js detectado automaticamente
